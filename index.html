What is Mobile Era?

Mobile Era is a conference about mobile development. Our topics: iOS / Android, Mobile web / PWA, Cross-platform / JS-to-Native, IoT / Hardware, Mobile UX / Accessibility

Mobile Era is organized by a team of enthusiasts across several technology companies in Oslo. We're passionate about developing mobile apps and want to see a kick-ass event in Oslo for people like ourselves. The organizing committee consists of developers who are also active in organizing open events, conferences and meetups in the Oslo developer community.

We want the crew, speakers, attendees to socialize!

Why to join Mobile Era?

As developers, we find that most of the exciting projects that we deal with have a small or large mobile component. Mobile technology is developing rapidly and every year, there are new techniques, technologies and frameworks we need to learn. Technology conferences can help us get started on the basic, but in order to excel in the mobile world, we need more advanced topics.

With this in mind, we created Mobile Era, Northern Europe's premier conference for mobile developers. At Mobile Era, we bring in world class speakers who go in detail on topics with iOS, Android and Mobile Web development. This is the conference where you can learn about tvOS, bluetooth, Android IoT, Progressive Web Apps, push notifications, machine learning, Swift, Cosmos DB, synching, hybrid apps and much more.

Mobile Era is made by the developer community, for the developer community. We want to make a valuable, content filled event for a reasonable price.
